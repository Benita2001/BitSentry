The Problem

Trading Agents Are Flying Blind

Most agents connect directly to exchanges with no safety layer between intention and execution.

01
⚠️

Black Box Agents

When your agent loses money, you don't know why. No decision trail. No accountability. Just a smaller balance.

02
💥

No Risk Controls

Agents trade without leverage caps, position limits, or loss circuit breakers. One bad trade wipes the account.

03
📉

Dead Strategies

Strategies degrade silently. Your agent keeps running a broken strategy while you watch the losses mount.

The Solution

5 Layers of Protection

Every trade passes through all 5 layers before touching Bitget.

01

Risk Guardian PROCEED / BLOCKED

5-layer pre-trade check. Leverage cap, position size limit, daily loss circuit breaker, consecutive loss throttle, and live market condition filter.

02

Audit Engine SHA-256 hash

Every decision logged to SQLite with SHA-256 integrity verification. Tamper-proof. Reproducible. Judges can verify independently.

03

Position Monitor GREEN / YELLOW / RED

Real-time safety ratings for every open Bitget position based on margin distance and unrealized PnL percentage.

04

Strategy Evaluator PERFORMING / DEAD

Rolling win rate over 7d and 30d windows. Detects strategy degradation before it destroys accounts.

05

Report Generator Telegram daily

Daily, weekly, and monthly summaries sent to Telegram automatically. Your agent reports to you, not the other way around.

Installation

Get Started in 4 Steps

BitSentry works alongside Bitget Agent Hub with zero conflict.

1

Install Bitget Agent Hub

bash
npx bitget-hub upgrade-all --target claude
2

Set credentials

bash
export BITGET_API_KEY="your_api_key"
export BITGET_SECRET_KEY="your_secret_key"
export BITGET_PASSPHRASE="your_passphrase"
3

Install BitSentry

bash
pip install bitsentry
4

Add to Claude Code MCP config

json
{
  "mcpServers": {
    "bitget": {
      "command": "npx",
      "args": ["-y", "bitget-mcp-server"]
    },
    "bitsentry": {
      "command": "npx",
      "args": ["-y", "@0xbeni/bitsentry-mcp"]
    }
  }
}
